# Hospital-Emergency-Room-Dashboard-Power-bi-
An interactive Power BI dashboard analyzing Hospital Emergency Room data. It tracks over 9,000 patients, average wait time of 35.3 minutes, and satisfaction score of 4.99. The dashboard provides insights by year, gender, race, and department to help improve hospital performance and patient care efficiency.
